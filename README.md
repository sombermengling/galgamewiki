## 📄 旗下学术期刊 —— 《Galgame研究》

**期刊官网：** [https://journal.galgame.it](https://journal.galgame.it)

《Galgame研究》是 Galgame Wiki 旗下的**在线学术期刊**，旨在为 Galgame 与视觉小说研究提供一个 **严肃、同行评议、开放获取** 的发表平台。我们不接受普通评测或感想文，只刊发具有**方法论、数据支持或理论深度**的原创研究。

### 已发表研究成果（部分）

#### 🔬 《素晴日玩家群体画像、评价分化与文化接受度研究》
- 综合批评空间、Steam、Bangumi 等多平台评分数据
- 通过聚类分析揭示“极化共识”现象——中央值高达 90 分，同时存在大量极端低分
- 结合社群讨论文本，解读不同玩家群体的接受心理

#### 🤖 《基于机器学习的视觉小说评价预测与情感分析》
- 首次将 SVM、随机森林、XGBoost 和 BERT 模型应用于视觉小说评分预测
- XGBoost 取得最优性能（RMSE=0.412）
- 情感分析揭示了玩家评论中的潜在情感维度

#### ⚔️ 《论〈Muv-Luv Alternative〉的狂气与救赎》
- 剖析该作在批评空间**历史第一**的排名背后的叙事策略
- 分析其“欺骗性”开局与后期宏大展开的反差结构
- 探讨“绝望—抗争—成长”主题如何构筑一部人类史诗

#### 🌧️ 《雨、星空与永不熄灭的梦——论〈星之梦〉的末世诗学与文明寓言》
- 以“雨”与“星空”为双核心意象，解构 Key 社短篇的文学性
- 提出“末世—信仰—传承”的三重对立结构
- 将作品置于日本后核灾文学语境下重新解读

## ❓ 常见问题（FAQ）

**Q：Galgame Wiki 和 VNDB、批评空间有什么区别？**  
A：VNDB 侧重英文资料和数据库，批评空间侧重评分排行，而 Galgame Wiki 侧重 **中文语境下的学术整理** 与 **深度内容**，包括理论、文化、争议话题和取景地，更接近一部“百科全书”而非单纯的数据库。

**Q：内容是否允许转载？**  
A：所有文字内容采用 **CC BY-NC-SA 4.0** 许可，允许非商业性转载，但须署名并相同方式共享。

**Q：如何参与编辑？**  
A：目前采用提交 Issue 或 PR 的方式（通过 GitHub），未来将开放在线编辑。详情参见网站“贡献指南”。

**Q：期刊是否收费？**  
A：完全开放获取，阅读和投稿均不收费。
