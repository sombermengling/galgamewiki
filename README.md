## ✨ 这是什么地方？

**Galgame Wiki** 是一个专注于 Galgame（美少女游戏）的中文百科与攻略社区[reference:0]。我们致力于为所有 Galgame 爱好者提供一个**免费、开放、持续更新**的知识库——从游戏资料、剧情解析到详细攻略，你想找的这里都有。

无论你是刚入坑的新人，还是想深挖某部作品的老玩家，这里都值得你收藏。

---

## 📚 网站主要提供什么？

### 🗂️ 游戏百科
收录了大量 Galgame 的详细信息，包括：
- 作品基本信息（日文名、英文名、开发商、发行时间等）[reference:1]
- 剧情简介与世界观设定
- 角色介绍与声优信息

目前已收录 **兰斯系列** 等多部经典作品，资料库仍在持续扩充中[reference:2]。

### 📝 文章与评测
不仅有客观的百科条目，还有玩家撰写的深度文章、作品评测与感想分享[reference:3]。在这里，你可以读到对一部作品最真实的解读。

### 🧭 游戏攻略
从主线流程到全CG收集，从路线选择到隐藏要素——攻略板块帮助你更好地通关和体验每一部作品[reference:4]。
